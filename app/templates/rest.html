Some Important Instructions for Using This API

The way to make a request is in the url as follows:

websitename.com/rest/request_type/tablename

Where 'request_type' is one of the following:

get -to pull information

add -to add a row to a table

update -to modify a field or row

delete -DISABLED by default

Ask site administrator to enable the delete if needed; however, it is recommended that any field deleting be done manually through the website form as it allows for more secure removal.
